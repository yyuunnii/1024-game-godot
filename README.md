# 1024 Game - Godot Version

A puzzle game inspired by 2048, rebuilt with Godot 4 for smooth animations!

## Features

- ✨ Smooth tile sliding animations
- 🎨 Elastic spawn effects
- 💥 Particle effects on merge
- 🏆 Win/Game Over animations with bounce
- 🎮 Keyboard controls (Arrow keys / WASD)
- 📱 Responsive design

## Play Online

[Play on GitHub Pages](https://yyuunnii.github.io/1024-game-godot/)

## Controls

- **Arrow Keys** or **WASD** - Move tiles
- **R** - Restart game

## Development

### Requirements

- Godot 4.2+

### Export to Web

1. Open project in Godot
2. Project → Export → Add → Web
3. Export to `docs/` folder
4. Push to GitHub for Pages deployment
